# 社交功能设计 v1 — 共同记忆撞上 & 表达引导

_基于 2026-05-23 产品讨论整理_

---

## 一、社交基础设施

- **Repo 点赞** — 对整篇演出记录点赞
- **一句话记忆点赞** — 社交钩子入口，对单条记忆标签独立点赞
- **Repo 评论** — 对整篇 repo 发表评论，支持单层回复（不做盖楼）
- **评论区默认折叠** — 条目详情页只显示入口，展开后查看更多
- **同场评论标识** — 同场用户的头像旁显示 🎸 标识，创造自然接触点
- **在场认证赞** — 同场用户的点赞带特殊标识

## 二、记忆表达引导

### 一句话记忆 — 句式模板
发布时提供填空式模板，用户选一种，也可自由模式：

- **A. 那一刻**：「____的那一刻，我____」→ 感性派
- **B. 比喻**：「这场演出像____」→ 比喻狂魔
- **C. 动作定义**：「我在这场演出____」→ 行动派
- **D. 自由模式** — 纯自由输入，AI 建议补充标签

**关联：** 个人主页展示「表达人格」标签

### 三维评分 — 形容词锚定
每个分值配描述锚点，评分时实时变化：

| 分值 | 乐队表现 | 音响效果 | 现场氛围 |
|------|---------|---------|---------|
| 1⭐ | 车祸现场 | 一团浆糊 | 全员养神 |
| 2⭐ | 不在状态 | 听了个寂寞 | 稀稀拉拉 |
| 3⭐ | 中规中矩 | 及格水平 | 礼貌性躁动 |
| 4⭐ | 全程高能 | 身临其境 | 真的燥起来了 |
| 5⭐ | 封神之夜 | 耳朵怀孕 | 掀翻屋顶 |

**关联：** 条目区展示「80% 的在场者认为当晚氛围『掀翻屋顶』」

### 照片标注
上传现场照片时打标签，反映「你的目光落在哪」：

- **台上**：吉他手 solo / 主唱嘶吼 / 鼓手爆裂 / 贝斯律动 / 灯光炸场
- **台下**：我在 pogo / 跳水瞬间 / 开火车经过 / 金属礼 / 合唱瞬间
- **环境**：场地全景 / 调音台视角 / 门口排队 / 散场后 / 门票特写

**匹配逻辑：** 照片标签一致 → 视觉审美匹配依据

## 三、「共同记忆撞上」三层级

| 层级 | 触发条件 | 信息暴露 | 破冰方式 |
|------|---------|---------|---------|
| **Lv1 撞演出** | 两人都标记同一场演出「看过」 | 仅计数（「有 X 个人也去了」） | 评论区 🎸 标识 |
| **Lv2 撞记忆** | 一句话记忆语义相似 / 或给对方记忆点了赞 | 半匿名展示对方记忆标签卡片 | 弹出「你们有相似的感受」：卡片 |
| **Lv3 撞 vibe** | 三维评分一致 + 记忆语义相似 + 审美重合度高 | 昵称 + 头像 + 重合度百分比 | 预设话术 + 「打个招呼」按钮 |

### 触发场景
- 发布 repo 后实时检测
- 浏览条目页时检测
- 发现页每日推荐
- **同场盲盒**：每日推一张卡片（匿名展示对方一句话记忆），共鸣→揭晓身份，无感→滑走，零社交压力

## 四、AI 能力

### AI 扩写 — 多声线选择
用户输入关键词后，AI 提供三种风格版本：

- **感官轰炸版** — 通感修辞，重描写
- **冷峻纪实版** — 克制冷静，只记事实
- **段子手版** — 黑色幽默，自嘲

**原则：** AI 不替你写，给你选。选择本身就是表达。

### AI 标签建议
- 自由模式下从内容中提取关键词
- 建议一句话记忆补充标签

## 五、数据模型变更

### 新增字段

**Repo 表：**
- `memory_template`: 用户选择的记忆模板类型 (A/B/C/D)
- `photo_tags`: 照片标注标签列表

**MemoryTag（一句话记忆）表：**
- `template_type`: 模板类型
- `template_filled`: 填空后的完整句子
- `likes_count`: 被赞次数

**Like 表：**
- `target_type`: repo | memory_tag | comment
- `target_id`
- `user_id`
- `is_same_show`: 是否为同场用户点赞

**Comment 表：**
- `parent_id`: 对评论的回复（nullable）
- `is_same_show_user`: 评论者是否为同场用户

**Match（匹配记录）表（新增）：**
- `user_a_id`, `user_b_id`
- `show_id`
- `match_level`: 1/2/3
- `similarity_score`: 审美重合度
- `memory_tag_a`, `memory_tag_b`: 用于卡片展示
- `matched_at`
- `status`: pending | connected | dismissed
