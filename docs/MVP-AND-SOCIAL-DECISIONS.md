# MVP 与深度社交：落地决策（已确认）

本文档记录与产品方对齐后的**可开工范围**。变更请同步 [FEATURE-BACKLOG.md](./FEATURE-BACKLOG.md) 与 `feature-backlog.csv`。

## 1. MVP 范围

| 决策项 | 结论 | 说明 |
|--------|------|------|
| 记录默认可见性 | **默认私密** | 公开到演出条目为显式二次操作。 |
| 记忆墙 | **MVP 必上** | 共同记忆的核心索引。 |
| **POA（整 Epic E）** | **MVP 不做** | 不含 EXIF 票根、记忆考古、身份背书、勋章；整体放入 **P2 / V1.5+**。 |
| Repo 列表排序 | **热门（高赞）/ 最新** | MVP 无「已验证」维度。 |
| 网易云偏好 | **V1 末或 V1.5**（P2） | 冷启动可先靠同场 + 手动标签。 |

**MVP 一句话**：找演出 → 发 Repo（默认私密）→ 公开到条目 → 记忆墙 → **点赞 + 评论** → 个人时间线。

**MVP 明确不做**：Epic E（POA）、私聊、小群/组队、AI 扩写（C7/C8）、小程序（J3）等见 backlog 中 P2+ 项。

## 2. 深度社交

| 阶段 | 能力 | 结论 |
|------|------|------|
| **MVP（V1）** | 点赞（G1）、评论（G2） | **必上** |
| V1 | 举报 / 申诉（G3） | P1，评论上线后尽快补齐 |
| **MVP 不做** | 半熟私聊（G4）、同场小群 / 组队（G5） | 见 P2 / P3 |
| V1.5+ | 半熟私聊 | 在 G3 跑通后再上 |
| V2 | 同场小群 / 组队想看 | 依赖关系链与通知 |
