# 产品功能清单（可导入排期）

**用途**：拆 Epic / Story；`depends_on` 为前置功能 ID（逗号分隔）。优先级：`P0` MVP、`P1` 首版后紧接、`P2` 增强、`P3` 远期。

**关联**：[MVP-AND-SOCIAL-DECISIONS.md](./MVP-AND-SOCIAL-DECISIONS.md)（范围默认）、[ARCHITECTURE-PRINCIPLES.md](./ARCHITECTURE-PRINCIPLES.md)（跨 Epic 约束）。
